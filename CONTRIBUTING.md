# Contributing to ARANGAM Space Kit

Thank you for your interest in contributing! This kit is designed to help people build their own cognitive operating systems. Your contributions make that mission stronger.

## What We Welcome

- **Translations**: Translate any template into your language
- **New examples**: Real-world, anonymized worked examples
- **Setup guides**: How to use the kit in Logseq, Roam, Bear, or other tools
- **Corrections**: Fix typos, improve clarity, update broken links
- **New templates**: Additional thinking structures that follow the anti-imitation standard

## The Anti-Imitation Standard

Every template in `/kit` must follow this standard:

1. Start with the "Steal This, Don't Copy It" header
2. Contain **guiding questions**, not pre-filled answers
3. Include an **Anti-Imitation Check** that forces the user to define their own context
4. Reference no SYNTHAI-specific examples (those go in `/examples` separately)

## How to Contribute

1. Fork the repository
2. Create a branch (`git checkout -b feature/my-template`)
3. Make your changes
4. Submit a Pull Request with a clear description

## Code of Conduct

Be respectful, constructive, and honest. We are building tools for human thinking; let's model good thinking in how we collaborate.
