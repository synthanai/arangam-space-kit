# Getting Started

**Three ways to use this kit. Pick the one that fits you.**

---

## Option A: Download and Use Anywhere

1. Go to [github.com/synthanai/arangam-space-kit](https://github.com/synthanai/arangam-space-kit)
2. Click the green "Code" button → "Download ZIP"
3. Unzip to your computer
4. Open the `/kit` folder in any text editor
5. Pick a template and start filling it in

**Best for**: People who want to try one or two templates quickly.

---

## Option B: Clone and Use in Obsidian

1. Clone the repo: `git clone https://github.com/synthanai/arangam-space-kit.git`
2. Open Obsidian → "Open folder as vault" → select `arangam-space-kit`
3. The `/kit` folder appears as a beautifully linked vault
4. Duplicate any template to start your own version
5. The internal links between methodology docs and templates work natively

**Best for**: Knowledge workers who use Obsidian for their second brain.

---

## Option C: Import to Notion

1. Download the ZIP (Option A)
2. In Notion, go to your workspace → Import → Markdown
3. Select the files from `/kit` that you want
4. Reorganize them in your Notion workspace

**Best for**: Teams already using Notion for collaboration.

---

## Option D: Use ARANGAM OS

1. Visit [arangam.space](https://arangam.space)
2. Create a free account
3. The templates from this kit are built into the app with AI-powered assistance
4. You get the same thinking structures, plus automated analysis and pattern detection

**Best for**: People who want the full automated cognitive operating system.

---

## Where to Start

If you've never used any of these templates before, start with these three:

1. **[CORE Template](../kit/identity/CORE-template.md)** — Know who you are
2. **[Signal Log](../kit/capture/signal-log.md)** — Start observing your world
3. **[Decision Record](../kit/decisions/decision-record.md)** — Write down your next important decision

That's it. Three files. Ten minutes each. You'll already be ahead of most organizations.
